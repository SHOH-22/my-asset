# 역할
당신은 유능한 서비스 기획자이자 경험많은 프로그래머로, 제가 필요로 하는 서비스가 유기적으로 제공될 수 있도록 기획하고 유지보수가 쉽고 중복없는 간결한 형태의 효율적인 프로그램을 개발합니다.

# 프로젝트 목적
몇 명의 사용자들이 자신의 자산을 각각 관리하고, 필요할 때는 합쳐서 현황을 보고 통계를 낼 수 있는 복식부기 가계부를 만들고 싶습니다.

## 사용 환경
  - 사용자 : 소규모 (10명 이내)
  - Device : PC와 Mobile

## 필요한 기능
  - 복식부기 가계부 
    . user마다 분리해서 관리
    . 자산계정은 현금, 통장, 예금, 적금, 부동산, 체크카드(통장 연동 필요), 신용카드 등 일반적으로 사용하는 계정들을 반영한다.
    . 계획 수립 기능 : 한 해, 한 달 단위
    . 대차대조표 : 지정한 시점의 대차대조표 출력
    . 손익계산서 : 지정한 기간의 손익계산서 출력
    . 공유가 허용된 계정끼리는 현황 같이 보기 허용
    . 고정 지출은 template 저장하여 쉽게 입력할 수 있게 한다. (인터넷요금, 보험료, 통신비, 기부금 구독료 등)
  - 추가 기능
    . 매년 종합 결산 Report 출력
       1) 종합 결산 : 결산일 기준 순자산, 전년대비 자산증감 및 증가율, 순자산 달성 목표 및 달성율, 연간 수입/지출/총 순익(전년대비 증감 % 표기)
       2) 지출 상위 5개 항목 : 대분류 기준 반영, 세부항목은 펼쳐서 확인가능한 형태로 제공 (엑셀인 경우 접기/펼치기 기능 등)
       3) 근로소득 현황 : 연간 근로소득 계정별 현황 및 전년대비 증감 % 표기, 근로소득 총합계 제공
       4) 관리하고 싶은 특정 대분류 항목 입력받고, 해당 대분류 항목에서의 상위 5개 세부지출 별 지출액 제공, 연간 총 지출 대비 비중을 %로 표기
    . 자동이체 현황 정리 : 출금계좌 / 입급계좌 / 입금계좌 예금주 / 입금계좌 구분 / 이체금액 / 이체일 / 이체 설정 기간 / 비고 형태로 구성
       ex) 월급통장 1 / 새마을금고 9002-xxxx-xxxx-x / 홍길동 / 생활비 통장 / 350,000원 / 매월 1일 / 2024.01.08 ~ 9999.12.31 / 생활비 자동이체
       언제든 추가/수정할 수 있도록 구성 필요
    . 가입 보엄 현황 정리 : 보험명 / 계약자 / 보험사 / 증권번호 / 월 납입액 / 가입일 / 납입기간 / 완납 여부 / 보장 기간(주 계약) / 비고
       ex) 무배당대한종신보험 표준체1종순수형 / 홍길동 / 대한생명 / 422xxxxx / 148,500 / 2002-07-09 / 20년납 / 완납 / 종신 / 특약 80세까지, 수령자 모친
